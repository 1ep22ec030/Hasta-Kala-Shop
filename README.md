# Hasta-Kala Shop

Hasta-Kala Shop is an Android-based micro-sales analytics and billing application designed for artisans and small craft businesses.

## Features
- Quick Billing System
- Inventory Management
- Sales Analytics Dashboard
- Best Seller Tracking
- Stock Alerts
- Offline Functionality
- Income Tracking

## Tech Stack
- Kotlin
- XML
- MVVM Architecture
- Room Database
- LiveData
- Coroutines
- MPAndroidChart
